# CLAUDE.md

> **먼저 [AGENTS.md](AGENTS.md) 를 읽으세요.** 이 프로젝트의 공유 컨텍스트(스택, 구조,
> 도메인, 디자인 토큰, 협업 규칙, 현재 상태)는 전부 AGENTS.md 에 있습니다.
> Codex 사용자와 공동작업하므로 AGENTS.md 가 **단일 진실 문서**입니다.

## Claude Code 작업 지침

- 항상 **AGENTS.md 를 먼저 읽고** 그 규칙·상태에 맞춰 작업한다.
- 프로젝트 상태/구조/결정이 바뀌면 **AGENTS.md(또는 docs/PROGRESS.md·DECISIONS.md)를
  갱신**한다. 컨텍스트를 CLAUDE.md 에 따로 중복 기록하지 않는다.
- API 변경 시 코드보다 **docs/API.md 를 먼저** 고친다.
- master 에 직접 커밋하지 않는다 — 브랜치 + PR.
- 작업 후 Codex 사용자가 알아야 할 내용은 AGENTS.md / PROGRESS.md 에 남긴다.
