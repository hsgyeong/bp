## DEC-0001 · 백엔드 스택: Spring Boot 4 + MyBatis + MySQL
- **날짜**: 2026-06
- **결정**: Java 21 / Spring Boot 4.0.6 / MyBatis / MySQL(`ssafy` 스키마) 사용.
- **이유**: 학습 범위에 맞춘 표준 조합. JPA 대신 MyBatis로 SQL 직접 제어.
- **영향**: 매퍼 XML은 `resources/mapper/` 하위, 응답은 camelCase 매핑.
