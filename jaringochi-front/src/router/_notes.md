### router/ 폴더
- 앱의 길 안내 지도를 모아두는 곳
- 어떤 주소에 어떤 화면을 보여줄지와 화면 이동 규칙에 관한 코드만 모아둠

```
src/router/
└─ index.js   ← 주소↔화면 표(routes) + 가드(beforeEach)
```

- "이 앱에 화면이 몇 개고 주소가 뭐지?" → router/index.js 하나만 열면 앱 전체 지도가 보여요
- 새 화면 추가 = 여기 routes에 한 줄 추가 (다른 데 안 건드림)
- 인증 가드 같은 이동 규칙도 여기 한곳에 → "로그인 안 하면 못 들어가는 화면" 규칙을 한 파일에서 관리


### 파일명이 index.js인 이유
import ... from '@/router'처럼 폴더명까지만 적으면, JS가 자동으로 그 폴더의 index.js를 찾는다. "폴더의 대표 파일" 관례.
