# 자린고비 가계부 — 화면 목업

손그림 + 텍스트 기획을 바탕으로 만든 HTML 화면 시안. 굴비 마스코트 + 골드/크림 톤.

## 보기
- 전체 갤러리: `index.html` 을 브라우저로 열기
- 화면 하나만: `index.html?only=N` (N = 화면 순서, 0부터)

## 화면 목록 (only 인덱스)
| N | 화면 |
|---|------|
| 0 | 로그인 |
| 1 | 홈 (굴비 메시지 · 주간 예산 사용률 · 월 수입/지출 · 최근 거래) |
| 2 | 가계부 (일자별) |
| 3 | 통계 (월 · 단순금액 / 꺾은선) |
| 4 | 회원가입 |
| 5 | 가계부 (달력) |
| 6 | 통계 (월 · 카테고리별 / 도넛) |
| 7 | 통계 (주 · 단순금액 / 막대+꺾은선) |
| 8 | 더보기 (설정) |
| 9 | 거래 등록 |
| 10 | 카테고리 설정 |
| 11 | 예산 설정 |
| 12 | 내 프로필 |
| 13 | 비밀번호 확인 · 회원 탈퇴 (바텀시트) |
| 14 | 비밀번호 수정 |

## 구성
- `index.html` — 전 화면 + 굴비 마스코트 SVG(`#gulbi` symbol)
- `style.css` — 디자인 시스템 (컬러 토큰, 카드, 탭바, 차트 스타일 등)
- 그래프는 전부 인라인 SVG (꺾은선/도넛/막대) — 캡처 시 깨짐 없음
